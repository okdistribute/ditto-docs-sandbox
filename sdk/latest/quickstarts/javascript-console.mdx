## Get Started

### 1. Clone the Repository

1. Open the terminal
2. Clone the repository from GitHub.
    ```script
    git clone https://github.com/getditto/quickstart
    ```
3. Navigate to the project root directory:
    ```script
    cd quickstart
    ```

### 2. Configure Ditto

#### Get Ditto App ID and Playground Token
<QuickstartConfigTitleAndAppIdBody />

#### Add Ditto Credentials

1. Copy the `.env.template` file at the root of the `quickstart` repo to `.env` and add your app ID and online playground token.
    ```shell
    cp .env.sample .env
    ```
2. Update the `.env` file with the App ID and Playground Token from your Ditto Application
    ```shell
    DITTO_APP_ID="REPLACE_ME_WITH_YOUR_APP_ID"
    DITTO_PLAYGROUND_TOKEN="REPLACE_ME_WITH_YOUR_PLAYGROUND_TOKEN"
    ```

### 3. Run the Application

1. Navigate to the project directory
    ```shell
    cd javascript-tui
    ```
2. Install dependencies
    ```shell
    yarn install
    ```
3. Run the Application
    ```shell
    yarn dev
    ```
