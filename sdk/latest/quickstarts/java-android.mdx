## Get Started

### 1. Clone the Repository
<QuickstartCloneTheRepoBody />

### 2. Configure Ditto

#### Get Ditto App ID and Playground Token
<QuickstartConfigTitleAndAppIdBody />

#### Add Ditto Credentials

1. Navigate to `app/src/main/java/com/example/dittotasks/MainActivity.java`
2. Update the following variables with the App ID and Playground Token from your Ditto Application
    ```java
    private String DITTO_APP_ID = "";
    private String DITTO_PLAYGROUND_TOKEN = "";
    ```

### 3. Run the Application

1. Launch Android Studio and open the `quickstart/android-java` directory
2. In Android Studio, select a connected Android device, or create and launch an Android emulator and select it as the destination, then choose the `Run > Run 'app'` menu item.
