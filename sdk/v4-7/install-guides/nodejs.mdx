## Prerequisites

Before you can install Ditto, you must have the following:

*   Ditto access credentials

*   [Node.js](https://nodejs.org/en/ "Node.js") active long-term support (LTS) version 18.x or 20.x

*   Linux, macOS version 11 (or later), or Windows version 10 (or later)

*   [Git](https://git-scm.com/downloads "Git")

*   Integrated developer environment (IDE)


<Info>
* To get your access credentials, see [Getting Playground Token Credentials](/v4-7/get-started/install-guides/nodejs#hK2s3Ld8Sj-Ti5snxwkQ1).

*   You can use any IDE you prefer; however, for optimal compatibility with the platform, Ditto recommends using [Visual Studio Code](https://code.visualstudio.com/download "Visual Studio Code").
</Info>


## Installing the Demo Task App

To install Ditto in your environment, install and set up Ditto, and authenticate your client identity with the Big Peer:

<Steps>
  <Step >
    Clone the [getditto > template-app-nodejs-todo](https://github.com/getditto/template-app-nodejs-todo/ "getditto > template-app-nodejs-todo") repository from GitHub:
  ```bash bash
  git clone https://github.com/getditto/template-app-nodejs-todo/
  ```
  </Step>
  <Step>
  Install the required dependencies:
  ```bash bash
  npm install
  ```
  </Step>
  <Step>
  Open the project in your IDE, and then open the `index.ts` file. For example, if using Visual Studio Code:
  ```bash bash
 code .
  ```
  </Step>
   <Step >
   In line 14 of the index.ts file, replace 'YOUR\_APP\_ID' and 'YOUR\_TOKEN' with your app ID and playground token. (See [Getting Playground Token Credentials](/ab-redirect-url?docId=U214QhYYMCxbV9jnamRFH&isHosted=true&queryTemplate=&shareableToken=&hostName=docs.ditto.live&docAnchorId=)﻿)
  ```bash bash
 code .
  ```
  </Step>
  <Step>
  In line 14 of the index.ts file, replace 'YOUR\_APP\_ID' and 'YOUR\_TOKEN' with your app ID and playground token. (See [Getting Playground Token Credentials](/ab-redirect-url?docId=U214QhYYMCxbV9jnamRFH&isHosted=true&queryTemplate=&shareableToken=&hostName=docs.ditto.live&docAnchorId=)﻿)
  ```TypeScript TypeScript
// replace YOUR_APP_ID and YOUR_TOKEN with p
ditto = new Ditto({ type: 'onlinePlayground', appID: 'YOUR_APP_ID', token: 'YOUR_TOKEN' })
  ```
  </Step>
    <Step >
Build the demo task app:
  ```bash bash
 npm run build
  ```
  </Step>
    <Step>
Interact with the demo task app and simulate real-time peer-to-peer replication by running the demo task app in two or more separate terminals (one for each virtual peer):
  ```bash bash
node index.js
  ```
  </Step>

  <Step>
  Using the commands that automatically appear in each of your terminals, add, remove, and modify documents as desired.
  </Step>
  <Step>
  If you want to observe your changes sync with the cloud-optional deployment, the [Big Peer](/v4-7/get-started/install-guides/undefined#0bpDy)﻿, open the data browser in the [Ditto portal](https://portal.ditto.live/ "Ditto portal").
  </Step>
</Steps>
